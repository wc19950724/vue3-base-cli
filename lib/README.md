# vue3+ts 脚手架工具

## Getting Started

To begin, you'll need to install `vue3-base-cli` globally:

```console
npm install -g vue3-base-cli
```

or

```console
yarn add -g vue3-base-cli
```

or

```console
pnpm add -g vue3-base-cli
```

**cli Description**

```markdown
vue3-base create <project-name>
```
